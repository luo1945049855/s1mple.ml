CSS

css三种写法，优先级（内联式>嵌入式>外部式）

  1. 内联式：写在html标签中。

    <p><span style="color:red">内联式</span></p>

  2. 嵌入式：写在当前文件中。

  3. 外部式：写在外部文件中。

    <link href="base.css" rel="stylesheet" type="text/css" />

注意：

1. 嵌入式写在外部式之后
2. 相同权值

继承

CSS的某些样式具有继承性，它允许样式不仅应用于某个特定的html标签元素，而且应用于其后代。

p{color:red;}具有继承性；
p{border:1px solid red;}不具有继承性。
